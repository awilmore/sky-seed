Sky Seed Project
{{env}} environment

This project can be used as a starting point for building MEAN stack applications that intend to use the Docker / Rancher setup.

  1. Copy everything to your new project folder
  2. Change all occurrences of skySeedProject to your own angular module name. This name is also used in package.json
  3. Change all occurrences of node-seed-project to reflect your own application name. This is used for mongo database names in the server/config/environment config files as well as for docker and rancher config in the docker folder
  4. Change the Sky Seed Project application name configured in client/index.html, server/newrelic.js and server/swagger.js to your own application name
  5. If you need a database, uncomment the mongoose.connect statement in server/server.js
  6. If you don't need a database, remove the mongoose code from server/server.js, remove the config from the various server/config/environment files and remove the mongoose module dependency from package.json
  7. Get dependencies with npm install
